## RAG RESUME ANALYZER
This project use retrieval argumented generation (RAG) to analyzer resumes and match them with job description.

## FEATURES
-Resume parsing from pdf
-Semantic search using embeddings
-FAISS vector database
-Matc score generation
-Skill gap analysis
## Tech Stack
-Python
-Hugging Face
-Langchain
-Faiss
## Workflow
1. Load resume
2. Split inti Chunks
3. Generate embeddings
4. Store in FAISS
5. Retrieve relevant data
6. Generate response using LLM
